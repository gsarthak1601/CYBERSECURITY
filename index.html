    Brute-Force Attacks

    Brute-force attacks utilize trial-and-error to guess passwords systematically. An automated script iterates through all possible character combinations (e.g. a, b, ..., z, then aa, ab, ...) until a match is found. Short or simple passwords can be cracked in milliseconds.

    Dictionary Attacks

    Instead of guessing random character combinations, a dictionary attack runs through lists of known popular passwords, leaks, and vocabulary words. Since many users pick common passwords, these dictionary lists compromise millions of accounts instantly.

    Symmetric Encryption

    Symmetric encryption secures sensitive data using a single secret key (or password) to lock and unlock files. If a user utilizes a weak password for encryption, an attacker can simply run a brute-force or dictionary script against the ciphertext to extract the decrypted file.

    The Math of Search Spaces

    Brute force attacks scale exponentially with length. The total search space size $S$ is calculated by raising the number of characters in the alphabet $C$ to the power of the password length $L$:

    S = C^L

    For 5 characters of Lowercase alphabet (26): $26^5 = 11,881,376$ possible combinations.

    Understanding Countermeasures

    Why do complex attacks fail on real production servers? Because servers employ **defensive policies**. Toggle various authentication policies below to observe how they affect a brute-force hacker script attempting to compromise an account.

    Active Defense Policies
    Rate Limiting

    Restricts requests to 5 attempts, enforcing a 3-second delay on subsequent attempts.

    Impact: -99% Attack Speed
    CAPTCHA Check

    Requires proof of humanity, inserting a forced 1.5-second processing delay per attempt.

    Impact: Adds heavy lag
    Account Lockout

    Completely freezes the user account for 1 hour after 3 consecutive failed entries.

    Impact: Shuts down attack
    MFA (Multi-Factor)

    Enforces a secondary token verification (such as SMS or app code).

    Impact: Attack Impossible
